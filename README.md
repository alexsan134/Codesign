# Codesigns
Copiar repositorio

```
git clone https://github.com/alexsan134/Codesign.git
```

Instalar Firebase

```
npm install -g firebase-tools
```

Iniciar sesión en Google Account

```
firebase login
```

Seleccionar carpeta de repo y despegar app

```
cd codesigns-master
firebase deploy 
```

La aplicación inicia desde la carpeta public
